# @supabase/utils

通用工具集，提供日志和辅助函数。

## 包名

`@supabase/utils`

## 导出入口

| 路径                     | 内容       |
| ------------------------ | ---------- |
| `@supabase/utils`        | 统一入口   |
| `@supabase/utils/logger` | 仅日志模块 |

## 模块

### logger — 结构化日志

基于 Pino 的日志工具，支持 JSON 输出和开发环境美化。

| 函数                        | 说明                         |
| --------------------------- | ---------------------------- |
| `createLogger(name, opts?)` | 创建带 `name` 标识的日志实例 |

## 使用规范

- 日志实例创建后全局复用，无需重复创建
- 开发环境自动启用 `pino-pretty` 美化输出
- 生产环境输出 JSON 格式日志，由日志收集系统处理
- 当前为桩状态（`export {}`），后续填充实现
