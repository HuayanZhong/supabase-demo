# @supabase/tools

AI 工具集，为 Agent 提供可调用的工具函数。

## 包名

`@supabase/tools`

## 导出入口

| 路径              | 内容     |
| ----------------- | -------- |
| `@supabase/tools` | 统一入口 |

## 工具清单

| 函数         | 说明                   | 状态 |
| ------------ | ---------------------- | ---- |
| `getWeather` | 获取指定城市的天气信息 | 桩   |

## 使用规范

- 每个工具函数为独立文件，纯函数，无副作用
- 新增工具在 `src/` 下新建文件并在 `index.ts` 中 re-export
- 当前为桩状态（`export {}`），后续按需填充
