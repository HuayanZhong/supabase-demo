# AMiner Data Search MCP

## 用途

学术数据查询工具，用于论文、学者、机构、期刊等学术数据查询。

## 触发条件

- 涉及论文查询
- 涉及学者信息
- 涉及机构数据
- 涉及期刊数据
- 学术数据分析

## 使用示例

```
任务：查询某篇论文的信息
→ 自动匹配：aminer-data-search MCP

任务：查找某位学者的研究成果
→ 自动匹配：aminer-data-search MCP

任务：分析某机构的研究方向
→ 自动匹配：aminer-data-search MCP
```

## 注意事项

- 数据来源需要标注
- 注意数据时效性
- 复杂查询可能需要多次调用
