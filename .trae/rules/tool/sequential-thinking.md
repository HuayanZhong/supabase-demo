# Sequential Thinking MCP

## 用途

顺序思考工具，用于复杂推理、多步骤思考。

## 触发条件

- 需要复杂推理
- 多步骤思考
- 逻辑分析
- 问题分解
- 决策支持

## 使用示例

```
任务：分析复杂问题的解决方案
→ 自动匹配：sequential-thinking MCP

任务：多步骤逻辑推理
→ 自动匹配：sequential-thinking MCP

任务：决策支持分析
→ 自动匹配：sequential-thinking MCP
```

## 注意事项

- 适用于复杂问题
- 简单问题不需要使用
- 结果需要验证
