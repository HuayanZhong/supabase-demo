# AutoGLM Deep Research MCP

## 用途

深度研究工具，用于深度调研、生成研究报告。

## 触发条件

- 需要深度调研
- 生成研究报告
- 综合分析
- 多维度研究
- 复杂主题探索

## 使用示例

```
任务：深度调研某个技术领域
→ 自动匹配：autoglm-deepresearch MCP

任务：生成市场分析报告
→ 自动匹配：autoglm-deepresearch MCP

任务：综合分析某个复杂主题
→ 自动匹配：autoglm-deepresearch MCP
```

## 注意事项

- 耗时较长
- 需要明确研究目标
- 结果需要验证
