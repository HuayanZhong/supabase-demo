# Filesystem MCP

## 用途

文件系统操作工具，用于文件读取、写入、编辑、目录操作。

## 触发条件

- 涉及文件读取
- 涉及文件写入
- 涉及文件编辑
- 涉及目录操作
- 批量文件处理

## 使用示例

```
任务：读取配置文件
→ 自动匹配：filesystem MCP

任务：批量重命名文件
→ 自动匹配：filesystem MCP

任务：创建目录结构
→ 自动匹配：filesystem MCP
```

## 注意事项

- 优先使用 Read/Write/Edit 工具
- 仅在需要批量操作或特殊文件操作时使用
- 注意文件权限和路径安全
