# AutoGLM Generate Image MCP

## 用途

图片生成工具，用于生成图片、文生图。

## 触发条件

- 需要生成图片
- 文生图
- 图像创作
- 视觉内容生成

## 使用示例

```
任务：生成产品示意图
→ 自动匹配：autoglm-generate-image MCP

任务：创建图标
→ 自动匹配：autoglm-generate-image MCP

任务：生成插图
→ 自动匹配：autoglm-generate-image MCP
```

## 注意事项

- 需要明确的描述
- 生成结果可能需要调整
- 注意版权和合规
