# AutoGLM Browser Agent MCP

## 用途

浏览器自动化工具，用于自动化浏览器操作、网页交互。

## 触发条件

- 需要自动化浏览器操作
- 网页交互
- 表单自动填写
- 网页数据采集
- 自动化测试

## 使用示例

```
任务：自动填写表单
→ 自动匹配：autoglm-browser-agent MCP

任务：采集网页数据
→ 自动匹配：autoglm-browser-agent MCP

任务：自动化测试流程
→ 自动匹配：autoglm-browser-agent MCP
```

## 注意事项

- 需要浏览器环境
- 注意隐私和安全
- 复杂操作可能需要多次调用
