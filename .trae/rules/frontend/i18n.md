# i18n 国际化约束

- 所有用户可见的文本必须通过 i18n 翻译，禁止硬编码语言字符串
- Locale code 必须使用 BCP 47 格式（`zh-CN`、`en`，连字符格式）
- Locale code 必须与 `@nuxt/ui/locale` 中对应 locale 的 `code` 字段一致
- 翻译文件必须放在 `packages/i18n/locales/` 下，按 locale code 命名
- 新增语言时，必须同步更新翻译文件、`index.ts`、`package.json` exports、`nuxt.config.ts` locales、`LocaleSelect.vue`
- 切换语言必须通过 `useI18n().setLocale(code)`，禁止手动拼接 locale 对象
