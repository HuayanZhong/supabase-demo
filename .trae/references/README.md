# References — 参考资料

## 职责

`references/` 存放项目中重要的参考文档，为 Agent 提供额外的背景知识和工作指引。

```
runtime/{domain}/router.md   ← 路由：决定加载哪些参考
          │
          ▼
references/*.md               ← 参考资料
```

---

## 文件结构

```
.trae/references/
├── README.md              # 本文档
└── subagent-guide.md      # Subagent 使用指引
```

## 行为约束

- 参考资料是辅助信息，不替代规则和规划文件的约束
- 参考资料由对应领域路由的资源表引用加载
