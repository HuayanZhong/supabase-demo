## 循环关系

```
              快循环（任务级修复）
               loop-governance.md
          ┌─────────────────────────┐
          │  Evaluation ❌          │
          │     → re-execute (≤3)   │
          │     → re-plan (≤2)      │
          │     → 人工介入           │
          └─────────────────────────┘
                    ↑
                    │ 每次迭代写入经验
                    ▼
          ┌─────────────────────────┐
          │  慢循环（治理级进化）     │
          │  evolution/             │
          │     收集 → 聚合 → 分析   │
          │     → 提案 → 应用 → 验证 │
          └─────────────────────────┘
                    │
                    ▼
          修改治理文件 → 下次任务生效
```
