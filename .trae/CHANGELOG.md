# Trae 治理架构变更日志

本文件记录 `.trae/` 目录下所有治理规则的变更历史。

## 格式说明

每条记录包含：

- **日期**：变更发生的日期（YYYY-MM-DD）
- **类型**：新增 / 修改 / 删除 / 重构
- **范围**：影响的文件或模块
- **描述**：变更的具体内容
- **原因**：为什么进行此变更

---

## 2026-07-16

### 安全改进

**类型**：修改  
**范围**：`hooks/inject-credentials.ps1`  
**描述**：移除明文密码注入，改为仅注入凭证状态标记  
**原因**：原实现将 Google 账号密码明文注入 AI 上下文，存在安全风险。改进后仅告知 AI "凭证已配置"，不暴露敏感信息。

### 新增 PostToolUse 验证

**类型**：新增  
**范围**：`hooks/post-tool-verify.ps1`  
**描述**：新增工具执行后自动 lint 验证 hook  
**原因**：Write/Edit 工具执行后自动运行 lint 检查，及时发现代码规范问题。

### 新增 Commit Message 验证

**类型**：新增  
**范围**：`hooks/validate-commit-msg.ps1`  
**描述**：新增 git commit 前自动验证 commit message 格式  
**原因**：确保所有 commit message 符合 Conventional Commits 规范且使用中文。
