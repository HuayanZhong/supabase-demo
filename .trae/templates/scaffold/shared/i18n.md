# i18n Entry 骨架

适用：新增或修改国际化翻译条目

## 涉及文件

| 语言     | 文件                               |
| -------- | ---------------------------------- |
| 简体中文 | `packages/i18n/locales/zh-CN.json` |
| 英文     | `packages/i18n/locales/en.json`    |
| 日文     | `packages/i18n/locales/ja.json`    |
| 韩文     | `packages/i18n/locales/ko.json`    |

## 条目格式

```json
{
  "{namespace}": {
    "{key}": "{value}"
  }
}
```

## 填充规则

- `{namespace}` — 领域/页面名，如 `goals`、`dashboard.home`
- `{key}` — 具体 key，如 `title`、`description`、`placeholder`
- `{value}` — 对应语言的翻译文案

## 新增条目步骤

1. 在 4 个语言文件中分别添加同一条目（key 相同，value 翻译不同）
2. 在 `packages/i18n/src/index.ts` 中确认新 key 被导出（如使用 `useI18n().t()` 自动查找则无需操作）
3. 在代码中使用 `t('{namespace}.{key}')` 引用

## 常用命名

```
.{namespace}.title       — 页面/区块标题
.{namespace}.description  — 描述文字
.{namespace}.placeholder  — 输入框占位文字
.{namespace}.empty        — 空状态提示
.{namespace}.error.{key}  — 错误提示
.{namespace}.action.{key} — 按钮/操作文字
```
