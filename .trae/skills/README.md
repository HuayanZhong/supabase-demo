# Skills — 技能包

## 职责

`skills/` 存放领域特定的技能包（Skill），为 Agent 提供专业知识库和可复用的能力。

```
runtime/{domain}/router.md   ← 路由：决定加载哪些技能
          │
          ▼
skills/{skill-name}/          ← 技能包：提供领域知识
          │
          ▼
Agent 使用技能辅助执行
```

---

## 文件结构

```
.trae/skills/
├── README.md                 # 本文档
└── ui-ux-pro-max/            # UI/UX 设计知识库
    ├── SKILL.md              # 技能说明与用法
    ├── data/                 # 知识数据（组件库、色板、图标等）
    └── scripts/              # 辅助脚本
```

---

## 当前技能包

| 技能              | 覆盖内容                               | 适用 Agent                      |
| ----------------- | -------------------------------------- | ------------------------------- |
| **ui-ux-pro-max** | 组件库选择、色板、图标、排版、样式规范 | ui-designer、frontend-architect |

---

## 行为约束

- 技能包与 MCP 工具互补：MCP 提供实时 API 调用，技能提供静态知识库
- 技能包在对应领域路由的 `资源映射` 表中声明
- 技能包优先于通用知识，避免 Agent 使用过时的经验
