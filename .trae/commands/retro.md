回顾整次对话的协作过程，覆盖**意图理解、执行效率、治理依从性、错误模式**四个维度，
输出复盘报告存档到 `.trae/snapshots/retro/`，持续追踪治理架构的演进趋势。

复盘范围：当前对话全量内容（含已压缩的上下文摘要）+ 历史复盘报告（如有）+ `.trae/agents/` Subagent 调用日志

详细步骤文件在 `.trae/commands/retro/` 目录下。执行到每一步时，先读取对应子文件获取详细指令。

## 设计约束

1. **只读操作**：本命令不修改任何项目文件，仅收集数据和分析输出
2. **全量分析**：覆盖已压缩和未压缩的所有对话内容，从系统提醒中提取压缩摘要
3. **基线存档**：每次报告保存到 `.trae/snapshots/retro/retro-YYYY-MM-DD-HHmm.md`，趋势可追溯
4. **安全操作**：所有改进建议仅输出报告，不自动执行；用户审查后决定是否落地

## 分析数据源

| 数据源                                    | 提供信息                                     |
| ----------------------------------------- | -------------------------------------------- |
| 对话消息                                  | 用户意图、AI 理解偏差、纠正次数、任务主题    |
| 系统提醒（System Reminder）               | 上下文压缩摘要、文件打开记录、lint 错误      |
| 任务日志（本会话已有）                    | 资源覆盖率、变更清单、质量验证结果、耗时偏差 |
| hooks 上下文注入                          | 哪些 hook 被触发、注入了什么规则指针         |
| 历史复盘报告（`.trae/snapshots/retro/`）  | 基线数据、趋势对比                           |
| Subagent 调用日志（`.trae/agents/logs/`） | 触发次数、Subagent 命中率、调度模式          |

## 分析范围图

```
整次会话（从开始到当前）
  ├─ 已压缩轮次 ─→ 系统提醒中的压缩摘要
  ├─ 未压缩轮次 ─→ 完整对话消息 + hooks 上下文
  ├─ 任务日志   ─→ 之前输出的结构化日志
  └─ Subagent   ─→ agent-invoke.log 调用记录
         │
         ▼
   综合分析
  ├─ 意图理解质量
  ├─ 任务执行效率
  ├─ Subagent 命中率（新增）
  ├─ 治理依从性
  ├─ 规则命中率统计
  └─ 错误模式识别
         │
         ▼
   输出报告 + 存档 + 改进建议
```

## 前置准备

| 路径                     | 缺失时处理               |
| ------------------------ | ------------------------ |
| `.trae/hooks.json`       | 降级：跳过治理健康检查   |
| `.trae/rules/`           | 降级：跳过规则依从性分析 |
| `.trae/snapshots/retro/` | 自动创建目录，本次为基线 |

## 执行步骤

每步执行前先读取 `retro/{file}` 获取详细指令，执行后输出统计摘要。

| #   | 步骤     | 详细指令文件           | 输出摘要格式                                                                                                 |
| --- | -------- | ---------------------- | ------------------------------------------------------------------------------------------------------------ |
| 1   | 数据收集 | `retro/01-collect.md`  | `[{步骤1 收集}] 消息 {N} 轮 \| 已压缩 {N} 轮 \| 工具调用 {N} 次 \| hook 触发 {N} 次 \| Subagent 调用 {N} 次` |
| 2   | 协作分析 | `retro/02-analyze.md`  | `[{步骤2 协作}] 意图理解 ✅/⚠️ \| 纠正 {N} 次 \| 无效工作 {N} 次 \| 耗时偏差 {N}% \| Subagent 命中率 {P}%`   |
| 3   | 健康检查 | `retro/03-health.md`   | `[{步骤3 健康}] hook 触发 {N}/{N} \| 规则引用 {N}/{N} \| 日志完整 {N}/{N} \| agent 日志 {N}/{N}`             |
| 4   | 模式识别 | `retro/04-patterns.md` | `[{步骤4 模式}] 问题 {N} 个 \| 高优 {N} \| 中优 {N} \| 低优 {N}`                                             |
| 5   | 输出报告 | `retro/05-report.md`   | 按模板输出完整复盘报告 + 存档到 snapshots                                                                    |
