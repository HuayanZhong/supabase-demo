# Step 5: 输出复盘报告

汇总前 4 步的分析结果，输出完整复盘报告并存档。

## 执行

### 5.1 报告生成

按以下模板输出完整 markdown 报告，并追加到对话末尾。

### 5.2 存档

- 确保 `.trae/snapshots/retro/` 目录存在（不存在则创建）
- 将报告同时写入 `.trae/snapshots/retro/retro-YYYY-MM-DD-HHmm.md`（使用当前时间的 UTC+8 时间戳）
- 存档文件与对话中输出的内容完全一致

### 5.3 存档清理

- 检查 `.trae/snapshots/retro/` 下的历史报告数量
- 保留最近 10 份，超出部分删除最旧的
- 确保不因长期使用产生过多存档
