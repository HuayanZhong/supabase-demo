# Commands — Trae 命名命令

## 职责

`commands/` 定义了 Trae IDE 的命名命令（Slash Commands），用户输入 `/命令名` 即可触发对应的自动化流程。

## 可用命令

| 命令             | 触发方式          | 功能                                                                    |
| ---------------- | ----------------- | ----------------------------------------------------------------------- |
| `sync-resources` | `/sync-resources` | 扫描 MCP 和 Skill 资源差异，自动同步到 registry.md 和各 governance 文件 |

## 使用方式

在 Trae IDE 对话中输入 `/命令名` 即可触发。命令执行后会输出日志说明操作结果。
